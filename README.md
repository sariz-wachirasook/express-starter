# All About The Starter Kit

## Node.js + Express + Prisma + MySQL

---

```shell
# (can also use yarn or npm)
pnpm install
pnpm schema:update
pnpm dev
```

## Key Features

---

### 1. User Management System

- crud users
- user login
- user register
- user refresh token
- user reset password
- user soft delete
- user roles and permissions (RBAC)

### 2. CMS System

- crud cms pages
- crud cms blogs
- slugify
- average reading time
- seo

### 3. Export System

- export users
  - csv
  - excel
- export pages
  - csv
  - excel

### 4. Email Templates

- welcome email
- reset password email

### 5. Cronjobs System

- every 1 minute
- every 30 minutes
- every 1 hour
- every 1 day
- every 1st day of week
- every 1st day of month
